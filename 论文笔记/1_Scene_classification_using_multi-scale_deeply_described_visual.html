Scene classification using multi-scale deeply described 
使用多尺度的场景分类深入描述


本文提出了MBVW（一种多尺度的视觉词汇）也即 深度视觉词汇统计 能够很好的表示场景并提高分类准确性

1、介绍

由于从图片中提取大量有用的信息较为困难，尤其时超高分辨率图像，虽然有一些学者提出了解决方法，但是效果不太理想，
很难从一些场景中根据语义信息进行分类（尤其实在相似度很高情况下），故 另一些学者提出了BoVW，虽有效的改善了场景分类，但是却忽略了视觉词汇的空间分布。利用CNN自学习能力，可以自动的提取信息，提出了MBVW，有效的改善了在视觉词汇空间分布上的分类问题。
