AID（航空图像数据集）: 用于航空场景分类性能评估的基准数据集

Introduction

1、像素有什么问题？

Blaschke and Strob认为在物体层面分析航拍图像更为有效，其中“物体”是指共享光谱的像素的局部区域或 纹理同性质

以往的方法是： 像素级和对象级分类方法都试图对底部的空中场景进行建模。通过聚合提取的光谱，纹理和几何特征来训练强分类器

空中场景分类旨在通过利用 空间布置 和 结构模式  的变化直接对场景建模来将空中图像分类为不同的语义类别 。

2、尽管近年来在空中场景分类方面取得了 许多令人兴奋的进展，但是又两个主要问题严重限制了空中场景分类的发展：

1）缺乏对现有方法的全面审查：也就是 没有控制好变量！
2）缺乏适当的基准数据集进行绩效评估

3、由于上述问题，在本文中，我们提出了一个全面的最新算法审查以及一个新的大型航空影像基准数据集（AID），以充分推进空中场景分类任务。

本论文的主要贡献 如下：
1）我们通过对场景分类方法的发展进行了清晰的总结，对空中场景分类进行了全面的回顾
2）我们构建了一个新的大规模数据集，即AID，用于空中场景分类
3）我们在新数据集上评估了一组具有各种实验方案的代表性空中场景分类方法
4）我们发布了所有基线算法实现的源代码，并将成为其他研究人员的通用工具

（注： AID---据我们所知，数据集是最大尺寸和图像的数据集具有较高的类内多样性和较低的类间差异性，可为研究界提供更好的数据资源，以评估和推进航空图像分析中的最新算法。）

Ⅲ------------------------ REVIEW ON }AERIAL }SCENE }CLASSIFICATION（空中场景分类回顾）

本节全面回顾了现有的航拍图像场景分类方法。 区别于以自下而上的方式解释航拍图像的像素/物体级图像分类，场景分类易于通过开发航拍图像的整体表示直接模拟空中场景

A：Methods Using Low-Level Visual Features

利用这种方法，假设可以通过低级视觉特征（例如，光谱，纹理和结构）来区分空中场景。

值得注意的是，具有低级视觉特征的场景分类方法在具有均匀结构和空间排列的一些空中场景上表现良好，但是它们难以描绘空中场景中的高度多样性和非均匀空间分布。

B：Methods Relying on Midlevel Visual Representations（方法依赖于中层视觉表征）

与依赖于低级视觉属性的方法相比，中级空中场景分析方法试图通过表示由提取的局部视觉属性形成的高阶统计模式来开发整体场景表示。

IV. BASELINE METHODS

在本节中，我们使用先前评估的低级，中级和高级场景描述来评估不同的空中场景分类方法。

A. Methods With Low-Level Scene Features

使用低级场景特征的航空图像分类方法通常首先将空间图像划分为小块，然后使用低级视觉特征（例如，光谱信息，纹理信息或结构信息）来表征补丁，最后输出补丁特征的分布作为场景描述符。
1) SIFT--------------------尺寸不变特征变换
2）LBP
3) Color nHistogram
4) GIST

2) SPM
3) Locality-constrained linear coding (LLC)
4) The pLSA
5) LDA
6) IFK
7) VLAD

D. Discussion

从上述实验结果中，我们可以总结出一些有趣但有意义的观察结果如下:

1）通过比较各种场景分类方法，我们可以观察分层性能，顾名思义：低级方法的性能相对较差，而高级方法在所有数据集上的表现都较好，这表明了高级方法的巨大潜力。

2）通过比较不同的场景分类数据集，我们可以发现，我们的新数据集比其他数据集更具挑战性，因为它具有相对较高的类内变异和较小的类间相异性。  此外，大量样本图像可以帮助更精确地评估各种方法。

上述观察结果可为我们提供非常有意义的指示，以便在更具挑战性的数据集上研究更有效的高级方法，以促进空中场景分类的进展。



VI. CONCLUSION
