摘要

本论文主要研究的是一种VHR 图像场景分类框架。  首先，采用VGG-Net模型从原始的VHR中提取特征，接着，我们选择由VGG-Net构建的完全连接的层，其中每层被视为分离的特征描述符，然后我们将它们结合起来构建VHR图像场景的最终表示。 采用判别相关分析（DCA）作为特征融合策略，进一步完善了从VGG-Net中提取的原始特征，与传统的特征融合策略相比，该融合方法成本更低。

第一部分

1) feature extraction with deep VGG-Net;
2) using the DCA approach to fuse the extracted features; 
3) using a support vector machine (SVM) as a classifier for VHR scene classification.

我们的框架是基于VGG-Net 模型

两个著名的特征融合策略：
1）Serial feature fusion 串行功能融合  ----将两组功能连接成一个功能。假设x和y分别是从具有p，q向量维度的输入图像中提取的两个特征，然后融合特征是z，其大小等于（p + q）。
2）Parallel strategy 并行策略 -- 它将两个特征向量组合成一个复数向量z = x + iy，其中i是虚数单位。
